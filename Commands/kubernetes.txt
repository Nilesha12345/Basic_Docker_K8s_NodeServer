To run from Code
kubectl create ns nodeserver
kubectl apply -f .\K8s\Node\ --namespace=nodeserver

To check deployment, pod and services
kubectl get all -n nodeserver



To delete Deployment
kubectl delete ns nodeserver 
   or 
kubectl delete -f .\K8s\Node\ --namespace=nodeserver


To run Code Github
For Namespace
kubectl create ns nodeserver
For Deployment
kubectl apply -f https://raw.githubusercontent.com/Nilesha12345/Basic_Docker_K8s_NodeServer/master/K8s/Node/deployment.yaml
For service
kubectl apply -f https://raw.githubusercontent.com/Nilesha12345/Basic_Docker_K8s_NodeServer/master/K8s/Node/service.yml
